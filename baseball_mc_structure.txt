## File Purposes

### Root Level
- **README.md**: Project description, setup instructions, usage
- **requirements.txt**: Python dependencies
- **.gitignore**: Ignore data files, cache, notebooks checkpoints
- **config.py**: Constants, parameters, settings
- **main.py**: Entry point for running simulations

### data/
- **raw/**: CSV/pickle files from pybaseball (git-ignored)
- **processed/**: Cleaned player stats with calculated probabilities

### notebooks/
- Jupyter notebooks for exploration, testing, visualization
- Not part of core codebase

### src/data/
- **scraper.py**: Functions to pull data using pybaseball
- **processor.py**: Clean data, calculate derived stats
- **league_averages.py**: Compute fallback distributions

### src/models/
- **player.py**: Player dataclass with stats and probabilities
- **probability.py**: ISO-based hit distribution, slash line decomposition
- **baserunning.py**: `advance_runners()` logic

### src/engine/
- **game_state.py**: GameState class/dataclass
- **inning.py**: `simulate_half_inning()`
- **game.py**: `simulate_game()`

### src/simulation/
- **season.py**: `simulate_season()`
- **batch.py**: `run_simulations()` for N iterations

### src/analysis/
- **metrics.py**: Calculate mean, CI, distributions
- **comparison.py**: Statistical tests between lineups
- **visualization.py**: Plotting functions

### tests/
- Unit tests for each major component
- pytest structure

### scripts/
- Standalone scripts for common tasks
- Can import from src/ modules
